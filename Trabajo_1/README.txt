# Instalar y activar ambiente virtual

## Instalación de virtualenv

pip install virtualenv

## Creación y activación virtualenv

### _Creación_:

virtualenv virtualenv

### _activación_ :

Linux: source virtualenv/bin/activate

Windows: virtualenv\Scripts\activate

Nota: en caso de estar usando la consola de git en Windows se debe realizar el comando con source virtualenv\Scripts\activate

# Instalación de dependencias

pip install -r requirements.txt

# Ejecutar el juego

python juego.py 

Nota: Activar el virtualenv antes de correr el juego
